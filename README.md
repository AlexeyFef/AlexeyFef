# Инженер по тестированию
## 	:raised_hand_with_fingers_splayed: О себе
Меня зовут Алексей, я тестировщик. Тестировал WEB приложения, мобильные приложения. Работал как в команде из четырех тестировщиков так и единственным тестировщиком на проекте. Владею различными техниками тест-дизайна, проводил функциональное тестирование, тестирование API, тестирование UI/UX, тестирование производительности, тестирование мобильных приложений. Изучаю Python, Selenium. Ниже представлена часть моих реализованных проектов.
